# NON-BILLING SAFTS' SUBIMISSION

### WINDOWS INSTALL:

```bash
python -m venv venv
.\env\Scripts\activate
pip install -r requirements.txt
python main.py
```

### LINUX INSTALL:

```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
