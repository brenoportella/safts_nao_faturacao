# SUBMISSÃO DE SAFTS DE NÃO FATURAÇÃO

### Para instalar (WINDOWS):

```bash
python -m venv env
.\env\Scripts\activate
pip install -r requirements.txt
python main.py
```

### Para instalar (LINUX):

```bash
python3 -m venv env
source env/Scripts/activate
pip install -r requirements.txt
python main.py
```
